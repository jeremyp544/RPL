## Cara Instalasi
1. Clone repository:
   ```bash
   git clone https://github.com/username/reponame.git
   ```
2. Masuk ke direktori project:
   ```bash
   cd reponame
   ```
3. Instal dependensi:
   ```bash
   npm install  # atau pip install -r requirements.txt
   ```
4. Jalankan project:
   ```bash
   npm start  # atau python main.py
   ```
